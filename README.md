# StepList AI Features Service

A [`dspy-cli`](https://pypi.org/project/dspy-cli/) service that manages the LLM-powered functions used by [StepList](https://steplist.app/home).

## Programs

- **Emojifier:** Accepts a list title and returns a single emoji.
- **Step Generator:** Provide a list title and optional description and get back an ordered array of steps.
- **List Generator from Website:** Given a URL, this program will fetch the webpage, review it, and generate a list title, description, and steps.
